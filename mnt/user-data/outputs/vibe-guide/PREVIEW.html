A Psychology-Driven Personalization Engine

Not a travel guide. A vibe matcher. Generate hyper-personalized micro-itineraries based on personality, mood, and intent.

This preview showcases the design system, user flow, and key features of the application.

Design Philosophy

Brutalist-Editorial

Bold typography, high contrast, intentional use of space. Combines the raw honesty of brutalism with the refined hierarchy of editorial design.

Distinctive Memorable

Typography First

Playfair Display (serif) for display creates emotional impact. DM Sans (sans-serif) for body provides clean readability.

Key Features

6 Personas

Foodie, Introvert, Artsy, Nature lover, History nerd, Broke college student — or type your own custom vibe.

8 Intent Modifiers

Cheap date, Avoid tourists, Walking route, Hidden gems, Photography, Spontaneous, 3-hour tour, Locals-only.

Smart Curation

2-4 carefully selected spots with personalized "why" explanations. Not 20 random suggestions — focused experiences.

2 Cities (Expanding)

Miami and Orlando with 48 total curated spots. Easy to add more cities by editing the backend.

Technical Stack

Frontend

React 18 + Vite for modern, fast development. Custom CSS with intentional styling — no Tailwind, no Bootstrap.

React Vite

Backend

Express + Node.js with in-memory data store. Simple, scalable, and lightning-fast (~5ms response time).

Express Node.js

Deployment

Multiple options: Vercel + Railway (easiest), Docker + DigitalOcean, or AWS serverless. Your choice!
